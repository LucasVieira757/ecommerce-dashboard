# Dashboard E-commerce

Projeto de análise de dados com PostgreSQL, Python e Streamlit.

## Tecnologias
- PostgreSQL (Supabase)
- Python
- Streamlit
- Plotly
- Pandas

## O que o dashboard mostra
- KPIs de vendas
- faturamento por mês
- top cidades
- faturamento por estado
- top produtos

- tabela detalhada
